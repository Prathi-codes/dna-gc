# dna-gc
# Beginner python tool that calculates the GC content of a given DNA sequence
In order to use: User must enter a valid DNA sequence 
**STABILITY** Adenine, Guanine, Cytosine and Thymine are nitrogenous bases in a DNA sequence. Adenine pairs with Thymine(DNA)/Uracil(RNA) through a double hydrogen bond whereas guanine pairs with Cytosine through triple hydrogen bond which is more stable.
**USE** studies show that bacteria with higher GC content are more stable and also that it facilitates autolysis thereby increasing the viability of the cell.
